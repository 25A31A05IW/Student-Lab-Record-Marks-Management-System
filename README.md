Student Grade Calculator in C++

Introduction

This project calculates the grade of a student based on marks.

Objective

To use conditional statements to assign grades.

Tools Used

- C++

Working

The user enters marks. The program checks the marks and assigns a grade.

Grading System

- 90 and above → A
- 75–89 → B
- 50–74 → C
- Below 50 → Fail

Result

The program displays the grade based on marks.
